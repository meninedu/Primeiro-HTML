Ferramentas Utilizadas

(Voltar)

Para melhor manuseio do código o uso de uma ferramenta de edição de uma pagina web, mais apropriada que um bloco de notas por exemplo, se faz necessário principalmente para um iniciante e tambem ajudando os mais experientes. Nesse caso iremos fazer uso do Visual studio code. Essa ferramenta ajuda na hora de usar Tags dando sujestões e uma visualização do código mais clara e organizada. Ainda conta com extenções para almentar a quantidade de funcionalidades. o Live Server por exemplo da um feedback instantaneo logo após o arquivo ser salvo, ajudando a saber se o dev está indo na direção desejada.

Inspetor de elementos

(Voltar)

Essa ferramenta é muito pratica para o estudo de uma página web já pronta. Podendo ver quais os caminhos usados pelo dev(s) para chegar naquele resultado. No caso do Google chrome basta aperta a tecla F12.

Estutura básica do HTML

(Voltar)

O HTML é composto por três pontos principais. O primero é a TAG html inicio e / html no final. Depois vem a head, que é coisas que ele irá carregar antes de levar ao usuario, como por exemplo o título da página. e por ultimo vem a body que é basicamente tudo que o usuario ira ver e interagir (se for o caso) dentro da página.

Tags

(Voltar)

A Tag é basicamente o comando que está sendo dado para o programa para que ele execute determinada função como alterar o tamanho de uma palavra por exemplo.

Atributos das Tags

(Voltar)

Atribustos são propiedades que uma Tag pode ter, sendo almuns mais genericos e outros mais específicos. Sendo por exemplo o text e number no caso de um campo a ser preenchido pelo usuario. Também para adicionar uma imagem e o tamanho que ela terá no site.

Textos

(Voltar)

Para os textos usa-se Tags específicas para a organizção em título, sub título, paragrafos etc.

Listas Ordenadas e Não Ordenadas

(Voltar)

A lista ordenada como o nome sugere serve para ordenar os itens da lista os sendo possível também criar outra lista dentro de um item da primeira.

Já na lista não ordenada ela não precisa seguir uma ordem, tendo os numeros sendo substituídos por um simbolo.

Links

(Voltar)

Os links server para direcionar o usuario a outra página na internet ou para um item na mesma pagina que ele está.
